# Employee CRUD API

## Tech Stack
- Spring Boot
- PostgreSQL
- JPA

## Features
- Create Employee
- Get All Employees
- Get by ID
- Update Employee
- Delete Employee

## Run
1. Create DB: employeedb
2. Update application.properties
3. Run application

## API
- GET /api/employees
- POST /api/employees
- PUT /api/employees/{id}
- DELETE /api/employees/{id}
